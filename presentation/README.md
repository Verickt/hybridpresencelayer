# Präsentation anschauen

## Setup

```bash
cd presentation
npm install
```

## Starten

```bash
npm run dev
```

Öffnet die Präsentation im Browser (standardmässig `http://localhost:3030`).

## Als PDF exportieren

```bash
npm run export
```

## Steuerung

- **Pfeiltasten** — Nächste / vorherige Folie
- **Leertaste** — Nächste Folie
- **F** — Vollbild
- **O** — Folienübersicht
