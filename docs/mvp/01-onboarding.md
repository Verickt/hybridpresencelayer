# MVP — Onboarding

All features built.

1. **Magic Link Auth** — No password, single-use time-limited links via email/QR
2. **Pre-filled Identity** — Name/email from organizer import, initials avatar with interest-based color
3. **Participant Type Selection** — Physical/Remote toggle, switchable mid-event
4. **Interest Tags** — Pick 3 from organizer-defined tag cloud
5. **Icebreaker Selection** — Optional, pick from pre-defined questions, shown on profile
6. **QR Identity Code** — Personal QR for scan-to-connect
7. **Progressive Identity Enrichment** — All phases: company/role prompt, intent free-text, registration data auto-import
8. **Intent & Availability Signals** — "Open to call", "Available after session", "Looking for [X]"
9. **Organizer Configuration** — Define tags, icebreakers, import participant list, open vs invite-only
