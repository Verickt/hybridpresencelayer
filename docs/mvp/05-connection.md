# MVP — Connection & Communication

## Build
1. **Mutual Match** — both ping → match notification with shared context, options: chat / call / save for later
2. **Instant Chat** — text only, 500 char limit, no read receipts, typing indicator, suggested icebreaker, event + 7 days retention
3. **3-Minute Video Call** — WebRTC, countdown timer, extend +3 min (max 9 min), icebreaker at start, no recording
4. **Connection Card Exchange** — auto-generated from profile, optional company/role/LinkedIn, includes interaction context
5. **Connection List** — chronological matches, shared tags, connection context, quick actions, filterable

## Stub (documented only)
- **Post-Event Contact Export** — vCard, CSV, email digest with context per contact
- **Physical↔Physical Location Hints** — "meet at the coffee area" suggestions
