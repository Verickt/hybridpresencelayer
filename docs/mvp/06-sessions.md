# MVP — Session Integration

All features built.

1. **Session Check-In** — QR scan (physical), tap "I'm here", or click "Join" (remote); auto-updates status/context badge
2. **Live Session Participants** — split physical/remote view, count, sorted by interest overlap, ping button, "people you might want to meet" highlights
3. **Session Q&A** — submit questions (physical + remote), upvote, mark answered, attributed with profile card, networking hook (ping question-askers)
4. **Session Reactions Stream** — live 💡👏❓🔥🤔 across both worlds, shared experience, reaction peaks as matching signals
5. **Post-Session Matchmaking** — 3-5 suggestions based on shared reactions/questions/tags, 15-min window, location hints (physical) or call prompts (remote)
6. **Session Schedule View** — list with times/speakers/descriptions, attendee count preview, tag overlap, 5-min reminders
7. **Organizer Controls** — create/edit sessions, per-session engagement metrics, toggle Q&A/reaction display, export analytics
