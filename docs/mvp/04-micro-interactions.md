# MVP — Micro-Interactions

## Build
1. **Ping** — one-tap interest signal, no message, 30-min expiry, max 10/hour, mutual ping → connection
2. **Session Reactions** — 5 reaction types (💡👏❓🔥🤔), live stream, semi-anonymous, shared moment signal for matching
3. **Icebreaker Prompts** — profile-based, context-based, icebreaker-based, and random; shown on profile view, after match, and in chat
4. **Share Interest** — ephemeral topic hand-raise during sessions/booths, expires after session
5. **Anti-Harassment** — rate limiting (10/hour, 30/day), block, report, mutual match required for chat, 3-ignore cooldown

## Stub (documented only)
- **Nudges** — system-initiated re-engagement prompts with timing rules and frequency controls
