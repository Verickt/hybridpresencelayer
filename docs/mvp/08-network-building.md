# MVP — Network Building

## Build
1. **Automatic Contact List** — every mutual match auto-saved, chronological, quick actions
2. **Context Per Contact** — how you met, shared interests, session context, interaction history, personal notes field
3. **Contact Card** — name, email, optional company/role/phone/LinkedIn, interest tags, icebreaker answer
