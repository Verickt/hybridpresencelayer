# MVP — Feedback Loop

## Build
1. **Explicit Signals** — ping (strong positive), mutual ping (very strong), "not interested" (strong negative), "later" (neutral), block (absolute negative)
2. **Cold Start** — new participants matched on interest tags only, first 3 suggestions diverse (tag-match, context-match, serendipity)
3. **Privacy** — feedback never shown to others, "not interested" is silent, behavioral data deleted 30 days after event, no cross-event profiling

## Stub (documented only)
- **Implicit Signals** — profile view duration, chat length, call extension, session/booth co-attendance
- **Per-Participant Adjustment** — personal scoring adjustments based on interaction history
- **North-Star Funnel Metrics** — suggestion→ping→match→conversation→follow-up funnel optimization
