# MVP — Booth / Stand Interaction

## Build
1. **Virtual Booth Presence** — booth card with name, company, description, tags, live visitor count (physical/remote), staff availability
2. **Booth Check-In** — QR scan (physical) or click "Visit Booth" (remote), updates context badge, creates visit record
3. **Booth Visitor Feed** — real-time split physical/remote, shared interests highlighted, visitors can ping each other
4. **Booth Staff Interaction** — see all visitors, initiate pings, send booth-wide announcements, priority indicator for high-relevance visitors
5. **Booth Ping** — two-way (visitor→staff, staff→visitor), creates standard mutual matches
6. **Booth Content** — links to resources, short pitch text, available to remote visitors
7. **Booth Discovery** — relevance-sorted directory, "Recommended for you", "Popular now", post-session suggestions
8. **Lead Capture** — name, email, tags, timestamp, duration, physical/remote, interactions, session context
9. **Lead Dashboard** — total visitors, interaction rate, exportable lead list (CSV), hot/warm/cold indicators
10. **Session-to-Booth Attribution** — track which session drove booth visits
11. **Privacy** — anonymous browsing option, consent notice on check-in

## Stub (documented only)
