# MVP — Organizer Dashboard

## Build
1. **Real-Time Overview** — active participants (physical/remote), connections made, interaction rate, cross-pollination rate, current activity
2. **Key Metrics/KPIs** — networking density, cross-pollination rate, match acceptance rate, engagement depth, time to first connection
3. **Session Analytics** — per-session check-ins, reaction rate/peaks, questions, post-session connections, networking score
4. **Booth Performance** — per-booth visitors, interaction rate, leads generated, comparison across booths
5. **Organizer Actions** — boost booth, highlight session, event-wide announcement, trigger serendipity wave, adjust matching weights
6. **Event Setup Flow** — 6-step wizard: event details, attendee import, content config, booth setup, matching tuning, review & launch
7. **Access Control** — organizer (full), booth staff (own booth), speakers (own session), participants (own stats only)

## Stub (documented only)
- **Participant Segments** — super connectors, active networkers, casual, observers, ghosts
- **Topic/Interest Clusters** — visual map of tag distribution and overlaps
- **Timeline View** — activity graph over time, peak moments, session correlation
- **Data Export** — PDF/CSV post-event report, anonymous raw data
- **Deeper Funnel Metrics** — suggestion→ping→match→conversation→follow-up funnel tracking
